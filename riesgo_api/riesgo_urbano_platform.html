Roadmap Deep Learning
1
Embedding de ubicación
Reemplazar target encoding por un vector denso de 32 dims aprendido end-to-end. Captura similitudes geográficas implícitas.
2
TabNet tabular
Red neuronal diseñada para datos tabulares con atención por columna. Interpretable y competitivo con XGBoost.
3
Ensemble DL + XGB
Stacking de predicciones: TabNet + XGBoost + meta-learner logístico. Objetivo: AUC > 0.93.
4
RAG multi-ciudad
Vector store por ciudad con informes de movilidad, noticias y estadísticas históricas para explicaciones contextualizadas.
